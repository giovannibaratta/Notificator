registerClient

abstract fun registerClient(eventSourceRequest: Set<ModuleRequest>, notificatorsRequest: Set<ModuleRequest>): String

Inizializza i moduli contenuti in eventSourceRequest e notificatorsRequest. Entrambi i set devono contenere almeno un elemento.

Parameters

eventSourceRequest - set di moduli da utilizzare come sorgente dati notificatorsRequest - set di moduli da utilizzare come notificatori

Exceptions

ClientRegistrationException - se si verifica un errore durante la registrazione

Return
Token univoco associato al cliente, da usare durante la deregistrazione o per le notifiche
