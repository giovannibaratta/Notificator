NotificatorInitializer(clientToken: String, notificatorsRequest: Set<<ERROR CLASS>>)

Ogni client è associato un NotificatorInitializer. Il NotificatorInitializer si occupa di inizializzare tutti i notificatori presenti in notificatorsRequest utilizzando i parametri oppurtuni. Se non si riesce a configurare anche solo un unico notificatore prima dello scadere del timeout viene lancita un'eccezione. Il notificatorInizializer può eliminare tutte le sottoscrizioni dai Notificator.

Parameters

clientToken - cliente a cui è associato notificatorsRequest - notificatori da inizializzare

Exceptions

BadRequestException - se uno dei notificatori da configurare non è disponibile TimeoutException - se non si è riuscito a configurare tutti i notificatori prima del timeout
